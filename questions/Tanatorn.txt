SOFTWARE PROCESS QUESTIONS

1. Describe about project planning
	Software project planning is task, which is performed before the production of software actually starts. It is there for the software production but involves no concrete activity that has any direction connection with software production; rather it is a set of multiple processes, which facilitates software production.
(http://www.tutorialspoint.com/software_engineering/software_project_management.htm)

2. What is the objective of code review?
	- To identify defects in any software work product.
	- Developers learning from each other, sharing knowledge and expertise.

3. What are the differences between TESTER and QA?
	QA : Prevent faults by inspecting & testing the process
	Tester : Detect faults by inspecting & testing the product
(http://sqa.stackexchange.com/questions/211/whats-the-difference-between-testing-and-quality-assurance)
