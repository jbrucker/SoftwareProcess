Software Process Question

1) Which one is not the outcome after the end of Elaboration Phase? (5)
	a) Most of Use case model has been completed (about 80%)
	b) Requirement scope of the overall project are understood
	c) Software Architecture Description
	d) Development plan for the overall project

Answer: b 

	Because you should understand all the requirement in the Inception Phase so that you can estimate the cost and risk of the project and discuss with customer. Then, you can accept the project and analysis the requirement in the next phase.

3) What is project estimation? (5)

Answer : It is a process to estimate various aspects of software product in order to calculate the cost of development in terms of efforts, time and resources. This estimation can be derived from past experience, by consulting experts or by using pre-defined formulas.

4) Define 3 advantages why we should perform code reviews? (5)
   [Describe 3 benefits of performing code reviews.]
Answer : 	
    •	Find and fix defects early.
	•	Code review often proves more cost effective than testing.
	•	Good mentoring/training/learning/team building, especially for new hires.
	•	Gives broader experience to team members.
	•	Makes it easier for one team member to maintain another's code.
	•	Improve overall quality of the code.
	•	Enforce coding standards.
